# Petrol search

- stitching images from UAVs/satellites
- NN for to search for petrol/oil spills


### Usage:

Up environment:

```bash
python3 -m venv env
source env/bin/activate
```
Install dependencies:
```bash
pip install -r requirements.txt
```

### Run:


```bash
python main.py
```
